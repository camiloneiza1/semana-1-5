Acerca de G-ED:

Nuestra plataforma es un avanzado Sistema de Gestión Académica el cual permite reducir el tiempo de procesos administrativos de rutina como transcribir calificaciones, calcular promedios generar automáticamente los informes para padres y docentes y entidades como el Ministerio de Educación y afines.

Visite nuestro repositorio de GitHub para más información.
